# Images et captures du portfolio

Ce dossier contient les ressources visuelles utilisées pour illustrer les différents projets du portfolio Aéroworld.

## Contenu du dossier

Les images peuvent inclure :

- captures d’écran des dashboards Power BI ;
- mockups UX/UI ;
- visualisations de données ;
- graphiques analytiques ;
- illustrations de projets Data & IA.

## Objectif

Ces visuels permettent de présenter rapidement :

- l’interface des tableaux de bord ;
- les résultats des analyses ;
- les choix de visualisation ;
- la qualité des livrables produits.

## Projets illustrés

Les captures concernent principalement :

- ONCFM — Détection de faux billets ;
- Vocal Weather ;
- DWFA — Drinking Water For All ;
- Étude de marché volaille.

### Détail des visuels

#### ONCFM — Détection de faux billets
- Courbe ROC comparant les modèles de classification ;
- Matrice de confusion du modèle final ;
- Capture CLI de prédiction sur données de production.

#### DWFA — Drinking Water For All
- Vue mondiale du dashboard Power BI ;
- Vue régionale OMS ;
- Vue nationale détaillée.

#### Vocal Weather
- Interface principale Streamlit ;
- Carte météo géolocalisée ;
- Prévisions météo sur 7 jours.

#### Étude de marché volaille
- Variance expliquée cumulée (ACP) ;
- Projection K-Means des clusters ;
- Top 10 des marchés recommandés.
