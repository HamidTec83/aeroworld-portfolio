# Conception des dashboards

Ce dossier regroupe les maquettes préparatoires utilisées pour concevoir les dashboards Power BI du portfolio Aéroworld.

Dashboards disponibles :

- Dashboard Profil Candidat  
- Dashboard Veille Data & IA  

Objectifs de conception :

- définir les KPI clés ;
- organiser les visuels ;
- améliorer la lisibilité ;
- assurer une cohérence UX/UI ;
- faciliter le storytelling décisionnel.

Compétences démontrées :

- design décisionnel ;
- data visualisation ;
- UX / UI dashboard ;
- communication visuelle.
